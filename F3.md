NIP-F3
======

Scrobbling
----------

`draft` `optional`

This NIP defines `kind:1073` as a _scrobble_ event. It has the following format:

```yaml
{
  "kind": 1073,
  "tags": [
    ["title", "<song title>"],
    ["album", "<song album>", "<optional variant>"],
    ["artist", "<musician or band>", "<optional variant>"]
  ]
  // other fields...
}
```
